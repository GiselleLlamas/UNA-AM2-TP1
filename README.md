# [cite_start]Somos Visibles [cite: 1]

[cite_start]**Somos Visibles** is an interactive multimedia project created to address the invisibility and discrimination that transgender people experience daily[cite: 4, 5]. [cite_start]The project invites users to interact constantly with the site, as interaction is a fundamental part of the conceptual narrative[cite: 7, 8].

## 🎯 Project Goals
[cite_start]The primary objective is to problematize the privileges of cis-heteronormativity by visibilizing the daily difficulties faced by trans individuals[cite: 22]. [cite_start]The experience treats these issues through both conceptual and explicit lenses[cite: 6].

## 🎨 Conceptual Framework
[cite_start]The project is built on the concept of **contrast**, specifically the tension between the "monochrome binary system" and the "multiplicity of colors"[cite: 52, 62, 68]. 

### Interactive Sections
[cite_start]The site is organized into sensory "stations"[cite: 111]:
* [cite_start]**Búsqueda Laboral (Work Search)**: A tactile experience that simulates a failed job search, where interaction with screen elements is intentionally blocked to represent systemic barriers[cite: 151, 153, 154].
* [cite_start]**Invasión (Dejar florecer?)**: A visual and tactile metaphor for personal growth, where users encounter obstacles that attempt to stop development, eventually revealing the message: *"No se puede tapar el sol con las manos"* (You can't cover the sun with your hands)[cite: 146, 147, 148, 149].
* **Posters**: This section contrasts mainstream media with real identity. [cite_start]Only posters featuring actual trans testimonies lead to video links; posters for mainstream films are non-interactive[cite: 156, 157, 158].
* [cite_start]**Mensajes (Messages)**: Users must search for hidden messages of empowerment within a sidewalk texture, symbolizing the presence of support networks that must often be sought out[cite: 161, 164, 165, 166, 167].
* [cite_start]**Oscuro (Penumbras)**: This auditory section resignifies identity by distorting audio clips from mainstream films that historically portrayed trans individuals as negative or harmful stereotypes[cite: 171, 173, 174, 175].
