# Somos Visibles 

**Somos Visibles** is an interactive multimedia project created to address the invisibility and discrimination that transgender people experience daily. The project invites users to interact constantly with the site, as interaction is a fundamental part of the conceptual narrative.

## 🎯 Project Goals
The primary objective is to problematize the privileges of cis-heteronormativity by visibilizing the daily difficulties faced by trans individuals. The experience treats these issues through both conceptual and explicit lenses.

## 🎨 Conceptual Framework
The project is built on the concept of **contrast**, specifically the tension between the "monochrome binary system" and the "multiplicity of colors". 

### Interactive Sections
The site is organized into sensory "stations":
* **Búsqueda Laboral (Work Search)**: A tactile experience that simulates a failed job search, where interaction with screen elements is intentionally blocked to represent systemic barriers.
* **Invasión (Dejar florecer?)**: A visual and tactile metaphor for personal growth, where users encounter obstacles that attempt to stop development, eventually revealing the message: *"No se puede tapar el sol con las manos"* (You can't cover the sun with your hands).
* **Posters**: This section contrasts mainstream media with real identity. Only posters featuring actual trans testimonies lead to video links; posters for mainstream films are non-interactive.
* **Mensajes (Messages)**: Users must search for hidden messages of empowerment within a sidewalk texture, symbolizing the presence of support networks that must often be sought out.
* **Oscuro (Penumbras)**: This auditory section resignifies identity by distorting audio clips from mainstream films that historically portrayed trans individuals as negative or harmful stereotypes.

## 🛠️ Technical Specifications
* **Languages**: Developed using HTML5, CSS3, and JavaScript.
* **Libraries**: Utilizes the **p5.js** library for interactive graphics and sensory logic.
* **Target Resolution**: Optimized for 1920 x 1080 px.

## 📚 Artistic References
The project draws inspiration from:
* **Yayoi Kusama**: For her themes of liberation and catharsis through the use of color.
* **Félix González-Torres**: For framing personal themes through figurative metaphors and expressive text.
